# Frontend guidance

- Read `context/current-state.md` before modifying the Android app.
- Run Gradle commands from `frontend/`.
- For ordinary frontend changes, verify with `./gradlew testDebugUnitTest` on Unix-like systems or `gradlew.bat testDebugUnitTest` on Windows.
- Update `context/current-state.md` after meaningful frontend changes.
- Add durable rules here only when they prevent a recurring mistake or capture repeated review feedback.
